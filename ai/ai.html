Answer Writing Tips for AI

  • Always start with definitions for fundamental concepts
  • Use diagrams for algorithms (search trees, game trees)
  • Include step-by-step examples for algorithms
  • Compare and contrast when asked (e.g., DFS vs BFS)
  • Mention real-world applications for theoretical concepts
  • For logic questions, show conversion step-by-step
